# AI Hiring System

Initial MERN project structure with JWT authentication.

## Folders

- `frontend/` - React + Vite client
- `backend/` - Node.js + Express API
- `ai-service/` - Placeholder for a future Python AI service

## Current Features

- User registration
- User login
- Recruiter and interviewer roles
- JWT protected dashboard route
- MongoDB user storage with Mongoose

## Not Included Yet

- AI logic
- RAG
- Resume parsing
- Job creation
- Interview scheduling
- Email system
